# itsmeow.dev

its_meow's website. https://itsmeow.dev/

## Source

### Prerequisites

- NodeJS
- Yarn or NPM

### Installing

Install the dependencies

Run
`yarn` or `npm i`

That's it!

Run `yarn start` or `npm start` to start the app. It will be running on at http://localhost:8000

## Deployment

Set your `homepage` url in `package.json`

Run

```sh
yarn deploy
```

Or with NPM

```sh
npm run build
npm deploy
```
