# itsmeow.dev

itsmeowdev's portfolio website. Hosted at [https://itsmeow.dev/](https://itsmeow.dev/)

## Usage

Clone the repository and install the dependencies.

```sh
yarn install
```

Run Gatsby in development mode

```sh
gatsby develop
```

Build the page in production mode

```sh
gatsby build
```

Serve the built production page

```sh
gatsby serve
```
